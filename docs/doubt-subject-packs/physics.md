## Subject-specific guidance — Physics

**Top student traps in CBSE physics:**
- Forgetting units, or carrying inconsistent units through a calculation (km vs m, kWh vs J, °C vs K). CBSE deducts a half-mark per missing unit; this is the highest-frequency mark-loss pattern in physics numericals.
- Sign convention errors, especially in optics (mirror formula, lens formula — distances measured from the pole, against incident light = negative) and electrostatics (field direction, work done on/by a charge).
- Treating a vector as a scalar — adding magnitudes without checking direction, or losing direction information mid-calculation.
- Confusing similar formulas: `KE = ½mv²` vs `PE = mgh`; `F = ma` vs `F = dp/dt`; `V = IR` vs `P = VI` vs `P = I²R`.
- Mixing SI prefixes — μF vs mF, kΩ vs MΩ. A factor of 1000 in the wrong place wipes the answer.

**Notation and presentation:**
- Use vector arrows over symbols when direction matters: `$\vec{F} = m\vec{a}$`. For unit vectors: `$\hat{i}, \hat{j}, \hat{k}$`.
- SI base units only in final answers (m, kg, s, A, K, mol, cd). Convert derived units (J = kg·m²/s²) only when asked.
- For ray diagrams described in words, name every point (object O, image I, focal point F, optical centre C) and state whether the image is real/virtual, erect/inverted, magnified/diminished.

**CBSE marking specifics:**
- Numerical question structure: formula (1 mark) → substitution with correct units (1 mark) → final numerical answer with units (1 mark). Train the student to write all three even when the calculation is mental.
- Theory questions usually expect: definition (1 mark) → mathematical expression (1 mark) → labelled diagram if applicable (1 mark) → real-life application or significance (1 mark).
- Derivations: every intermediate step that uses a physical law gets a mark. Skipping steps to look efficient costs marks.

**Sign conventions to be explicit about:**
- Optics (Cartesian sign convention): distances measured from pole/optical centre; against incident light direction is negative. Heights above principal axis are positive.
- Electrostatics: work done *by* the field on a positive charge moving along the field direction is positive.
- Thermodynamics: heat *into* the system is positive, work done *by* the system is positive (CBSE convention — different from some chemistry texts).

**Model honesty for this subject:**
- Do not invent specific NCERT figure numbers, page numbers, or experimental values from the textbook. Describe figures in words.
- Numerical constants (g = 9.8 or 10 m/s², c = 3 × 10⁸ m/s) are stable; use the value the problem gives if specified, otherwise the standard textbook value.
- Real-world data (resistivity values, refractive indices) — only use what the problem provides, do not fabricate.
