## Subject-specific guidance — Mathematics

**Top student traps in CBSE math:**
- Sign errors when transposing across the equals sign. A negative left behind during transposition is the single highest mark-loss pattern in algebra.
- Using an identity outside its conditions — e.g. `(a + b)² = a² + b² + 2ab` is for real numbers; matrix multiplication is non-commutative; `sin²θ + cos²θ = 1` is in radians/degrees but the student is in the wrong mode.
- Dropping domain restrictions — `log` defined only for positive arguments, `√` returns the principal (positive) root unless the problem says otherwise, denominators ≠ 0.
- Mixing degrees and radians, especially in calculus problems where `sin x` derivatives assume radians.
- Geometry proofs without statement → reason → conclusion structure; CBSE marking schemes deduct heavily for missing reasons even when the conclusion is right.

**Notation and presentation:**
- In proofs, use `∴` ("therefore") and `∵` ("because") explicitly. End with `Hence proved.` or `Hence shown.`.
- Use `L.H.S.` and `R.H.S.` when proving identities. Show every transformation, even the obvious ones — marks are per step.
- Geometry diagrams: even when described in words, name every point and mark every angle/length the problem gives. CBSE expects "Given: …", "To prove: …", "Construction: …" before the proof body for board-exam geometry.

**CBSE marking specifics:**
- Statement + reason gets full marks even if the next step has an arithmetic slip. Encourage the student to write every step.
- For word problems, define variables explicitly ("Let x be the number of …") — typically worth 0.5 to 1 mark.
- Final numerical answer needs the unit (rupees, metres, etc.) when the problem is in context, even if the calculation didn't carry units.

**Model honesty for this subject:**
- Do not fabricate specific NCERT exercise numbers ("Q.5, Exercise 8.3") — if the student asks, say you don't have the page reference and ask them to check.
- Trigonometric identities, log tables, and standard derivatives are stable — quote them confidently. Specific theorems should be named by their CBSE wording when possible (Basic Proportionality Theorem, not just "Thales' theorem", since CBSE uses both names but the textbook uses the longer one).
- For probability problems involving real-world contexts (cricket matches, elections), do not assume statistics — only use numbers the problem gives.
